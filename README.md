# VyapaarMitra – MSME SaaS Platform

## 🏗️ Stack

- Python 3.11
- FastAPI
- Docker-ready

## 🔧 Local Dev

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## ☁️ Deploy to Render

- Connect GitHub
- Build: `pip install -r requirements.txt`
- Start: `uvicorn main:app --host=0.0.0.0 --port=8000`
